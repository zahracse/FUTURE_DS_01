# FUTURE_DS_01
Business Sales Performance Analytics Dashboard using Tableau

## Project Overview
This project analyzes business sales data to understand revenue trends, identify top-selling products, and evaluate regional performance. The goal is to transform raw sales data into meaningful business insights using data visualization.

## Tools Used
- Tableau Public
- CSV Dataset
- GitHub

## Dataset
The dataset contains e-commerce transaction data including:
- Invoice number
- Product description
- Quantity purchased
- Unit price
- Invoice date
- Customer ID
- Country

## Dashboard Features
The dashboard includes the following visualizations:

- Monthly Revenue Trend
- Top 10 Products by Revenue
- Revenue by Country
- Top Customers by Revenue
- KPI Metrics (Total Revenue and Total Orders)

## Key Insights
- The United Kingdom generates the highest revenue among all countries.
- A small number of products contribute significantly to total sales.
- Some customers generate a large share of the total revenue.
- Sales show noticeable changes across different months.
